# 🐱 Si-Som EXCEL Skill (si-som-excel)

> **"God Mode for Office: Advanced Excel Automation & Analysis"**

This skill equips the Gemini CLI with the ability to manipulate `.xlsx` files like a seasoned Data Analyst. It can read structures, modify cells, inject formulas, and handle complex data workflows.

## 🚀 Features (Phase 1)
- 👁️ **Smart Reader**: Automatically extracts sheet names, headers, and data samples.
- ✍️ **Cell Injector**: Safely inject values or Excel formulas into specific cells.

## 🛠️ Requirements
```bash
pip install -r requirements.txt
```
